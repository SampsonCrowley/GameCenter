TETRIS
=========
A canvas rendered JS tetris game

# [Play Game](https://htmlpreview.github.io/?https://github.com/SampsonCrowley/tetris/blob/master/index.html)

# Controls

### W, Space, Click : Rotate 90

### Q, Up : Rotate -90

### A, Left : Strafe Left

### D, Right : Strafe Right

### S, Down : Speed Drop
